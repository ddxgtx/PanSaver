# PanSave 🚀

**PanSave** 是一款基于 Vue 3 + Flask + SQLite 架构的**跨平台网盘订阅与自动转存系统**。旨在帮助用户轻松实现对百度网盘分享链接（RSS、网页正则、单链接等）的实时监控、差异对比以及事务性级联转存。

---

## ✨ 核心特性

### 1. 📂 统一的网盘适配层与文件浏览器
* **扫码登录**：支持安全的百度网盘二维码扫描授权，自动换取并加密存储登录凭据。
* **文件浏览器**：提供美观的文件管理器 Dialog，直连网盘接口，支持上下级目录跳转、面包屑导航以及文件属性格式化。
* **游客模式隔离**：目录树分析爬取和转存会话完全隔离，完美绕过百度防爬的重定向循环和 `errno: -7` 报错。

### 2. ⚡ 多维转存策略与数据安全
* **增量同步 (Incremental)**：计算两次扫描的快照差异，仅下载/转存新增或修改过的文件。
* **安全覆盖 (Safe Overwrite)**：执行事务性的多步覆盖：**转存临时目录 ➔ 校验完整性 ➔ 备份正式目录 ➔ 部署新版本 ➔ 失败回滚**。保障转存过程中即便发生网络崩溃，数据也绝对不会丢失。
* **版本归档 (Version Archive)**：根据发布日期或版本标识，归档转存为独立的版本历史文件夹。
* **分片级联优化**：支持 500 个文件一组的自动分批转存，突破网盘单次转存数量限制；采用 Fast mkdir 优化，目录级联创建效率提升 2 倍。

### 3. 🎯 智能解析与规则引擎
* **一键智能解析**：在新建任务表单中，直接粘贴包含链接和提取码的整段网盘分享文本，系统会自动进行双重正则提取并实时回填。
* **高级过滤规则**：支持利用强大的正则表达式对文件名或路径进行 Include / Exclude 过滤。
* **重命名引擎**：支持使用正则表达式对转存后的文件名执行批量正则替换。
* **规则测试工具**：前后台打通规则验证机制，实时反馈文件名匹配和改名效果。

### 4. 🛡️ 工业级运行稳定性
* **异步日志队列**：使用内存队列机制异步串行写入 `operation_logs` 数据库，消除 SQLite 锁冲突、锁屏死锁，确保任务日志 100% 不丢失。
* **例行日志收割**：每日凌晨 3:00 自动唤醒，清理 30 天前的旧任务痕迹与日志，防范数据库无限膨胀。

## 🛠️ 技术栈

* **前端**：Vue 3 + TypeScript + Element Plus + Vite
* **后端**：Flask + SQLite3 + APScheduler + Loguru
* **打包**：PyInstaller 单文件跨平台构建

## 🚀 快速开始

### 1. 环境准备
确保您的电脑中已安装 Python 3.8+ 及 Node.js 环境。

### 2. 本地运行

#### 后端启动：
```bash
# 安装依赖
pip install -r requirements.txt

# 启动后台服务 (默认绑定 http://127.0.0.1:5000)
python web_app.py
```

#### 前端开发启动：
```bash
cd frontend
npm install
npm run dev
```

### 3. 生产编译与打包
如果您想生成 Windows 独立的可执行单文件程序（EXE）：

```bash
# 1. 编译前端静态资源
cd frontend
npm run build

# 2. 运行打包脚本进行 PyInstaller 封包
cd ..
python desktop/build.py
```
编译完成后，最终的运行程序将输出在 `dist/web_app.exe`。

## 🔒 隐私与安全性约束
* 用户的 `BDUSS`、`STOKEN` 以及分享密码在数据库中均以加密（PBKDF2 动态派生密钥对称加密）状态存储。
* 日志系统自动对网盘 Cookie 及敏感提取码进行过滤脱敏，防止凭证外泄。
